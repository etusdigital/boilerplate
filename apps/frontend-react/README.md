# Frontend React

React frontend for Etus Boilerplate.

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS v4
- React Router v6
- Zustand (state management)
- Auth0 React
- react-i18next
- Axios

## Development

```bash
# Install dependencies
pnpm install

# Run dev server (port 3002)
pnpm dev

# Build for production
pnpm build

# Run tests
pnpm test
```

## Port

This app runs on **port 3002** (different from Vue on 3000).
